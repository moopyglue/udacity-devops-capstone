Instructions

Before you start start make sure you can reach this page from both the screen you want to show your eye(s) on and the smart phone you want to control the eye with.

Click on LEFT or RIGHT eye link above and you will be presented with a qrcode.

Scan the QR code with your smartphone camera and it will sugest a link, follow that link and it link the screen and smart phone together and it will bring up the control screen.

NOTES:

  • the control screen is desigend to be used with the phone in landscape mode.
  • the eye only starts moving once you touch the touch control screen
